# Model Card: Systematic Reviewer AI (SR-Gemma4)

## 1. 모델 세부 정보 (Model Details)
- **베이스 모델**: gemma4 (Google)
- **구동 환경**: Ollama (Local Execution)
- **주요 용도**:
  - 의료 문헌 검색 쿼리(PICO) 생성
  - 제목/초록 기반 연구 적합성 판별 (Screening)
  - 전체 텍스트에서 비뚤림 위험(RoB) 및 대상/중재 결과 데이터 추출
  - 종합 결과(Synthesis) 리포트 텍스트 생성

## 2. 의도된 용도 (Intended Use)
- **Primary Use Cases**: 체계적 문헌고찰(Systematic Review) 과정을 수행하는 의료 연구자, 대학원생을 위한 연구 보조.
- **Out-of-Scope Uses**: 이 시스템의 출력물은 최종 임상적 결정을 내리는 근거로 단독 사용되어서는 안 되며, 반드시 인간 전문가의 상호 검증 및 수정 과정이 수반되어야 함.

## 3. 평가 데이터 및 결과 (Evaluation Data & Results)
- **RoB(비뚤림 위험) 평가 정확도 리포트** (2026-02 테스트 기준)
  - 대상 문헌: 노인 임플란트 관련 34편
  - 랜덤 배정, 눈가림, 결측치 처리 등 5개 도메인 평균 일치도: 85%
  - *참고: 비정형 텍스트에 내포된 저자의 의도를 파악하는 영역(예: 선택적 보고)에서는 상대적으로 보수적인 평가(High Risk 도출 경향)를 보임.*

## 4. 윤리적 고려사항 (Ethical Considerations)
- 데이터 프라이버시: 모든 LLM 처리가 로컬에서 이루어지므로, 사용자가 입력하거나 수집한 PDF 문헌이 외부로 유출되지 않음.
- 편향(Bias): 베이스 모델에 내재된 언어 편향이 존재할 수 있으며, 특정 인구통계학적 특성에 대한 의학적 결과 도출 시 검증이 필요함.
